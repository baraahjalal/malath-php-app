**Malath (ملاذ)** is a dedicated web platform designed to be a safe haven for women to grow, find support, and nurture their true potential. Built with a focus on femininity, community, and holistic well-being, the platform provides resources, community engagement, and personal development tools.

> *"ملاذكِ الآمن للنمو والدعم" - Your safe haven for growth and support.*

---

## 🚧 Project Status

**This project is currently in its VERY FIRST PHASE of development.** 
The foundational UI/UX, core routing, and database structures are being actively designed and built. Many features are planned and works-in-progress.

## 🌟 The Four Pillars of Malath

The platform is built upon four core pillars designed to cover all aspects of a woman's life:

1. 🌿 **صحتكِ أولاً (Your Health First)**: Integrated programs for physical health, nutrition, and a healthy lifestyle.
2. 💖 **سكينتكِ النفسية (Your Psychological Peace)**: Counseling sessions and safe communities for self-expression and mental well-being.
3. 📖 **نوركِ الروحي (Your Spiritual Light)**: Balanced religious and spiritual content to enhance spiritual values.
4. ⭐ **طموحكِ بلا حدود (Your Ambition Has No Limits)**: Academic and professional empowerment to help achieve life goals and develop passion.

## 🛠️ Planned Features & Architecture

Based on the current architecture, the following modules are being developed:

- **User Authentication:** Secure `login.php` and `register.php` for personalized experiences.
- **Personal Dashboard:** (`dashboard.php` & `profile.php`) A centralized hub for users to track their progress, saved articles, and community interactions.
- **Articles & Content Hub:** (`article.php`) Rich, informative content aligned with the four pillars.
- **Safe Community Forums:** (`community.php`) A secure, moderated space for women to connect, share experiences, and support one another.
- **Help & Support:** Comprehensive FAQ (`faq.php`) and Contact (`contact.php`) portals.

## 🚀 Getting Started (For Developers)

To run this project locally, you will need a local server environment (like XAMPP, MAMP, or Laragon).

1. **Clone the repository:**
   ```bash
   git clone https://github.com/baraahjalal/malath-php-app.git
   ```
2. **Move to your server directory:**
   Move the cloned folder into your `htdocs` (for XAMPP) or `www` (for WAMP) directory.
3. **Start your local server:**
   Ensure Apache and MySQL (if a database is added later) are running.
4. **Access the application:**
   Open your browser and navigate to `http://localhost/malath-php-app/`

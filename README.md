# 🌸 Malath (ملاذ) — A Safe Digital Haven for Women

![Status: Under Construction](https://img.shields.io/badge/Status-Under_Construction-ff69b4?style=flat)
![Language: PHP](https://img.shields.io/badge/PHP-Core-777BB4?logo=php&style=flat)

Malath (ملاذ) is a dynamic web platform created to provide women with a safe, supportive, and judgment-free digital space.
It is designed for those navigating pressure, isolation, and self-growth—offering a place to express, heal, learn, and thrive.

> *"ملاذكِ الآمن للنمو والدعم" — Your safe haven for growth and support.*

## 💭 Why Malath Exists

In a fast-paced digital world, truly safe spaces—especially for women—are rare.
Many platforms lack privacy, empathy, and cultural awareness.

Malath was built to change that.
A space where women can speak freely, share experiences, and receive support—without fear of judgment or toxicity.

## 🌟 Core Pillars

Malath is built around four essential pillars that support a woman’s full journey:

- 🌿 **صحتكِ أولاً (Your Health First)**
  Promoting physical health, nutrition, and balanced living.
- 💖 **سكينتكِ النفسية (Your Psychological Peace)**
  Safe expression, emotional support, and mental well-being.
- 📖 **نوركِ الروحي (Your Spiritual Light)**
  Meaningful spiritual and religious content with balance and depth.
- ⭐ **طموحكِ بلا حدود (Your Ambition Has No Limits)**
  Academic, career, and personal growth support.

## 🛠️ Features & System Modules

### 👤 User System
- Secure registration & login system
- Session management with role-based access (User / Admin)

### 📝 Content & Community
- Create, edit, and delete posts (CRUD system)
- Share experiences, stories, and thoughts
- Interactive comment system for community support

### 🗂️ Categorized Content
- Organized sections (Health, Psychological, Spiritual, Academic, Social)
- Admin-controlled category management

### 🛡️ Admin Dashboard
- Full platform moderation
- Content control (edit/delete inappropriate content)
- User management system (including ban functionality)

## 🧱 Architecture & Technologies

- **Backend:** PHP (Core Logic & Server-side Processing)
- **Frontend:** HTML5, CSS3, JavaScript (Responsive UI & Interactivity)
- **Database:** MySQL (Data Storage & CRUD Operations)
- **Architecture Pattern:** MVC (Structured & Scalable Design)
- **Security:**
  - Password Hashing
  - Input Validation
  - Session Management

## 🗺️ Project Structure (Key Pages)

- `index.php` — Landing page
- `login.php` / `register.php` — Authentication
- `home.php` — Main feed
- `category.php` — Content by category
- `post_details.php` — Full post view + comments
- `add_post.php` — Create content
- `admin_dashboard.php` — Admin control panel
- `manage_users.php` — User management

## 🚀 Getting Started

To run locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/baraahjalal/malath-php-app.git
   ```
2. Move the project to your server directory (`htdocs` / `www`)
3. Start Apache & MySQL
4. Open: `http://localhost/malath-php-app/`
